# ADVANCED-ENCRYPTION-TOOL

## OBJECTIVE:

To build a robust file encryption and decryption tool using AES-256 encryption algorithm.

## TECHNOLOGIES USED:

-Python3.

-Cryptography library.

-AES-256(CBC mode).

-Script key derivation.

-PKCS7 padding.

## FEATURES:

-Encrypt files using AES-256.

-Decrypt encrypted files.

-Secure key generation using password.

-Salt and IV protection.

-User-friendly menu interface.

## HOW IT WORKS:

-Generates secure key using Scrypt.

-Creates random salt and IV.

-Encrypts file using AES-256(CBC Mode).

-Stores-salt+IV+encrypted data.

-Decrypts using some password.

## HOW TO RUN:

### 1.Install dependency:
pip install cryptography.
### 2.Run:
Python advanced_encryption_toll.py
### 3.Choose encrypt or decrypt option.

## TESTING:

Create a sample file (e.g., test.txt) and verify encryption/decryption.

## SECURITY NOTE:

Incorrect password win results in decryption failure.
